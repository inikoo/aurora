static Class inputEx.JsonSchema

Namespace containing utility functions for conversion between inputEx JSON format and JSON Schema based on "Json Schema Proposal Working Draft": http://groups.google.com/group/json-schema/web/json-schema-proposal-working-draft The proposal is still under discussion and the implementation is very minimalist.
